# 🚀 Realm Arena

Realm Arena is a modern **real-time multiplayer gaming platform** where users can create rooms, invite friends, and play games like **Cricket 🏏** and **Ludo 🎲** with a smooth and engaging experience.

Built with performance, scalability, and clean UI in mind.

---

## 🌟 Features

### 🔐 Authentication

* Email & Password login
* Google OAuth login
* Secure session management
* User profiles with stats

---

### 🎮 Multiplayer System

* Create private rooms
* Join using room codes
* Real-time gameplay sync
* Player status (online / in-game)

---

### 🏏 Cricket (Multiplayer)

* 2-player real-time matches
* 20-over & 3-over modes
* Toss system
* Score tracking (runs, wickets, overs)
* Match result system

---

### 🎲 Ludo (Multiplayer)

* 2–4 players
* Dice roll system
* Token movement & capture
* Safe zones & win conditions

---

### 🎨 UI / UX

* Clean **light theme (Google-style)**
* Smooth animations
* Responsive design (mobile + desktop)
* Modern card-based layout

---

| Layer        | Technology         |

## 📦 Installation & Setup

### 1️⃣ Clone the repo

```bash
git clone https://github.com/Cosmoxpi/realm-arena.git
cd realm-arena
```

---

### 2️⃣ Install dependencies

```bash
npm install
```

---

### 3️⃣ Setup environment variables

Create a `.env` file:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_key
```

---

### 4️⃣ Run locally

```bash
npm run dev
```

---

### 5️⃣ Open in browser

```
http://localhost:8080
```
